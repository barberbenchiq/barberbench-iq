# BarberBench IQ — Founder Beta 0.4

A multi-tab barber business intelligence prototype with:
- Main business score dashboard
- Service-level pricing and time profitability
- Retail margin and opportunity analysis
- Red / amber / green performance status
- IQ Coach action list
- Goals, team insights and growth forecast
- Browser-local data persistence

## Run
```bash
yarn install
yarn dev
```

## Vercel
Framework: Next.js
Install command: `corepack enable && corepack prepare yarn@1.22.22 --activate && yarn install --network-timeout 600000`
