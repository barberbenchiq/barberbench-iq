# BarberBench IQ — Founder Beta 0.3

A mobile-first barber business dashboard built with Next.js.

## Features
- Dashboard with revenue, expenses, profit, average ticket and business health
- Daily sales entry
- Client book and follow-up status
- Expense tracking
- Service pricing and hourly earning analysis
- IQ Coach recommendations
- Local browser storage (no database required for founder testing)
- Responsive desktop and mobile layouts

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Production build
```bash
npm run build
npm start
```

## Important beta note
Founder Beta 0.3 stores data in the browser on the current device. Cloud accounts, secure authentication and database sync are planned for the next infrastructure phase.
